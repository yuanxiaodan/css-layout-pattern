<!-- html -->
                    <div class="one-column-vertical-center-5-wrap">
                    <div class="one-column-vertical-center-5"></div>
                    </div>
                    <!-- css -->
                   .one-column-vertical-center-5-wrap{
                    height:100px;
                    position:relative;
                    }
                    .one-column-vertical-center-5{
                    position:absolute;
                    top:50%;
                    transform:translateY(-50%); 
                    background:#ccc;
                    height:50px;
                    width:200px;
                    }

优点：居中元素不会对其他的产生影响。 缺点：transform属于css3内容，兼容性存在一定问题，高版本浏览器需要添加一些前缀。
